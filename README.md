2026年2月より製作開始

このwebappでできること:

このrandom-dartsはランダムに特定のナンバーを表示し、それを狙うというダーツの練習用webappです。
（現在はhtmlでformは作っている状態ですが、サーバーサイドでデータベースをいじってないのでformが送られない設定になっています。DBを理解してきたらまた実装します)追記　2026 5/11


このwebappを作成した理由:

僕は大学１年の後期にatcoderという競技プログラミングに友達に勧められハマりました。その過程でpythonでatcoderに参加するようになり、その過程でpythonを学びました。せっかくのpythonなのでatcoderだけではなく何か形に残るものを作ろうと思い、好きなダーツの練習が出来るアプリを開発しようと思いました。
ダーツなどの運動はランダムの的を狙うと上達率が上昇するという研究があるらしくランダムでナンバーを出す、かつ初心者に向いているこのアプリを制作しました。



使用技術:

python(flask),html,css,git(hub),render




このwebappを作るうえで大変だったこと：

これまでPythonは競技プログラミング用途でしか使用した経験がなく、Webアプリ開発の知識はほぼゼロからのスタートでした。
・FlaskのルーティングやMVC構造の理解
・開発環境の構築
・HTML / CSSとの連携方法
・デプロイ時のエラー対応
これらがかなり難しくAIや外部情報を活用しながら試行錯誤を繰り返して改善しました。
わからない部分をそのままにせず、調査と実装を繰り返して解決した点が最も大きな学びでした。



工夫した点:

・Flaskのルーティング構造を学習し、シンプルなWebアプリとして設計
・バックエンドとフロントエンドの役割分離を意識して実装
・UIは過度に複雑にせず、練習用途としての使いやすさを優先
・開発初期は知識がほぼ無い状態からスタートし、youtubeやqiita、本（flask）を活用しながら独学で実装を進めたこと




今後の改善点:

DBの実装、DBを使い成功率や苦手なナンバーの表示
